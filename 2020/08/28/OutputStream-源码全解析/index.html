发表于 2020-08-28 分类于 JDK 阅读次数： Disqus：

OutputStream 与 InputStream 对应，用于操作字节流。OutputStream 也是一个抽象类，实现了 Closeable 接口与 Flushable 接口。该类将 Java 层缓存的数据输出到 IO 设备。

OutputStream 抽象类中定义的方法

以上三个 write 方法时 OutputStream 抽象类中的核心方法。write 方法将数据输出到 Java 层级的 IO 缓冲区中。

flush 方法定义将 Java 缓冲区的数据强制刷到 IO 设备中。需要子类重写该方法。

close 方法关闭 stream 并且释放相关的资源。需要子类重写该方法。
