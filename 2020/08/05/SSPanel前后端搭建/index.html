SSPanel前后端搭建

发表于 2020-08-05 阅读次数： Disqus：

本来并不想写这篇博文，但是由于网上的教程很多很杂，所以还是自己重新写一篇用于记录搭建过程。中间包括记录了一些踩过的坑。

工具准备：

  1. VPS
  2. 域名

前端搭建

前端将使用 bt 来搭建。

前期准备

因为 bt 面板需要开放特定的端口，所以自己的 vps 首先需要打开这几个端口，并重新加载 firewall。

安装 bt 面板以及安装环境

执行一下命令开始安装。

中间过程可能会让你输入 y。输入就完事了。安装过程需要的时间挺久的。安装完之后会给出初始的密码，如下图所示。

安装完之后，根据给出的 IP 和端口，就可以访问 bt 面板了。首次登陆需要输入用户名和密码，并勾选同意用户协议。

选择左边的进行安装，同样等待其安装完成。

网站设置

安装完成后，在 bt 面板中进行操作。点击左侧导航栏的网站，创建一个新的站点。同时创建一个数据库。

在网站设置中设置伪静态规则。

回到终端界面，将 SSPanel-Uim clone 到网站目录下。

这里注意拉取的是 dev 版本，因为我试验过，master 版本存在问题，而 dev 版本是正常的。因为最新的 dev 版本不稳定，有 bug，要切到 ed15711277183014dbd564b514aafcc1f1b17788 。
dev 分支已修复 bug，可以正常使用。

设置 git 的配置。

安装成功之后回到 bt 面板中，设置网站目录并清除防跨站攻击，并把运行目录改为 /public。

安装 PHP 包，需要等待一段时间。此处出现错误正常，因为需要设置 PHP 的环境，禁用相关的函数，参看下一小节。

PHP环境设置

设置取消禁用函数，[system, putenv, popen, proc_open]。
