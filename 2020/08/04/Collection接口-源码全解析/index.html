Collection接口 源码全解析

本文 Collection接口 源码解析基于 JDK14-GA 版本。

Collection接口 的完整源码注释可以参看本人 Github 中的 JDK14 仓库

Collection 接口是日常常用的 Set/List/Queue 的父类接口。需要注意的是，另外一个常用的 Map 接口与 Collection 接口是两两独立的。

Collection 是存储元素(element) 的容器。存储在容器中的元素在不同具体的容器中有不同的属性，这些元素可以是有序的，也可以是无序的，还可以是不能重复的。一般不通过直接实现 Collection 接口来定义新的容器，而是通过实现 Collection 接口的子接口来实现。实现具体的容器的时候一般需要至少定义两个构造方法，一个是无参的构造方法，用于创建一个空的容器。另一个是带一个参数，参数类型为 Collection 类型的构造方法，可以将容器内的元素进行复制，用于构造出指定类型的容器。
